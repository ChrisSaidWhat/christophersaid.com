Selectors Exercise Instructions:

The following five web pages contain exercises to help you develop your skill using various types of CSS selectors to modify and control the look of the content. Open each web page in a text editor. Follow the instructions written in the comments of the embedded CSS style. View your results by opening the web page in a browser.

The image at the bottom of each pages displays how your web page is to appear when you are finished.

When completed upload this exercise to your host server, create a link to this finished exercise on your WDV205 homework page.

You can practice this exercise online at Selectors Tasks
